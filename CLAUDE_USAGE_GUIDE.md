# 📖 CLAUDE.md 使用ガイド

## 🎯 概要
CLAUDE.mdは、Whisperで文字起こしされたトランスクリプトから、プロフェッショナルなマークダウンレポートを自動生成するための包括的な仕様書です。

## ✨ 主な機能

### 1. インテリジェント分析
- **トピック自動検出**: 収益モデル、成長戦略、事例などを自動分類
- **重要統計の抽出**: 数値や実績を自動検出してハイライト
- **アクションアイテム生成**: 実践可能な行動指針を初級・中級・上級別に整理

### 2. スクリーンショット戦略
- **重要シーンの自動検出**: 講師紹介、衝撃的統計、ビジュアルデモなど
- **重み付けアルゴリズム**: シーンの重要度を0.8〜1.0でスコアリング
- **適切な配分**: レポート全体で均等に分散配置

### 3. 高品質レポート生成
- **Gemini Ultimate Generator統合**: 100点品質の要約生成
- **プロフェッショナルな書式**: エモジ、見出し、引用を使った視覚的に魅力的なレイアウト
- **instagram_seminar_summary.md品質**: 実証済みの高品質テンプレート

## 🚀 使用方法

### 基本的な使用
```bash
# トランスクリプトからレポート生成
python process_transcript.py \
  --input output/transcripts/seminar.json \
  --output output/reports \
  --video videos/seminar.mp4 \
  --screenshots 10
```

### 既存システムとの統合
```bash
# VideoTranscriptAnalyzerの完全フロー
python video_transcript_analyzer.py \
  --input video.mp4 \
  --report-type gemini \
  --generate-markdown true
```

## 🎨 レポート構造

### 生成されるレポートの標準構成
1. **エグゼクティブサマリー** - 核心メッセージを2-3文で
2. **最重要ポイント** - 引用形式で最も重要な洞察
3. **講師プロフィール** - 信頼性を確立
4. **核心メッセージ** - 番号付きの主要セクション
5. **収益/実績セクション** - 具体的な数値
6. **成長ストーリー** - ビフォーアフター
7. **実践アクション** - レベル別行動指針
8. **まとめ** - 成功への鍵
9. **参考情報** - メタデータ

## 🔍 トラブルシューティング

### よくある問題
1. **品質スコアが低い**
   - 解決: Geminiのトークン割り当てを増やす

2. **スクリーンショットが欠落**
   - 解決: タイムスタンプの精度を確認

3. **構造が不適切**
   - 解決: トピッククラスタリングアルゴリズムを見直す

4. **要約が不完全**
   - 解決: セグメント処理を確認
